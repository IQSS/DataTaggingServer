About This Questionnaire

This is a proof-of-concept DataTags model of the unemployment benefits chapter of the Israeli National Insurance Law. It was created as a demo for ESPAnet Israel 2017 by Rotem Medzini and Michael Bar-Sinai. This interview is an attempt to see if the DataTags models, which were designed to handle dataset handling policies, can be used to model entitlement-based public policies as well.

Warning: This is a Proof of concept only
This interview and the model it is based on were developed as a proof of concept and should not be considers a legal advice of any sort.
